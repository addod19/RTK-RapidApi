## Getting Started

To get a local copy up and running follow these simple example steps.

1. ``` git clone https://github.com/addod19/RTK-RapidApi.git ```
2. ``` cd RTK-RapidApi ```
3. ``` git checkout -b crpto```
4. ``` yarn i ```
5. ``` yarn start ```
